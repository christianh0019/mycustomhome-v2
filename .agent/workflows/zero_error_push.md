---
description: Zero Error Push Protocol
---
1. Run `npm run build` to verify the codebase compiles.
// turbo
2. View the output of the build command. If it fails, STOP and fix the errors.
3. Run `git status` to see pending changes.
4. Run `git add .` to stage changes.
5. Run `git commit -m "feat: [describe changes]"` (Prompt user for message if not provided, or infer from context).
6. Run `git push origin main`.
